What observations and insights can you make here from the data?

Several conclusions can be drawn from examining the NBA coach performance information that is presented. Identifying the coach who finished the 1985 season with the most wins highlights an exemplary example in coaching brilliance for that particular year, so igniting an investigation into the tactics or team dynamics that enabled such accomplishment. Even a thorough summary of team performance is given by showing the average number of wins (which combines regular season and playoff triumphs) for each Team ID (TID). This gives insights into the general competitiveness of various teams throughout time. Determining the coach with the greatest number of teams coached reveals trends in coaching mobility and flexibility, inspiring additional research into whether managing several teams is associated with general coaching success or particular accomplishments.

What are you using to design your web interface?

The web interface is designed using HTML. Plotly Express is used for creating interactive data visualisations. JS is used to handle the interactions and work with Plotly Charts.

What is the architecture for the web page?

The architecture is as follows: Client side: HTML, CSS, Java Script | Server side: Flask, Plotly Express | Data: CSV

How are you reading the data?

Data reading is done using the pandas library in python.
